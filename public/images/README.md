# Images Folder

Add your images here!

## Required Images:

- `hero.jpg` - Your photo for the hero section (recommended: 600x600px, square)
- `avl-trees.jpg` - Image for AVL Trees research section (recommended: 600x600px, square)
- `rocket1.jpg`, `rocket2.jpg`, `rocket3.jpg`, `rocket4.jpg` - Four rocket images for Asteria 1 project collage (recommended: square format)

## Image Tips:

- Use JPG or PNG format
- Square images (1:1 ratio) work best
- Recommended size: 600x600px or larger
- Keep file sizes reasonable (< 500KB each)

The website will automatically use these images once you add them here!
