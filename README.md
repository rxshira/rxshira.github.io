# Shira Rubin - Personal Website

## Tech Stack

- React 18
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React (icons)

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Design System

The website uses a refined color palette inspired by natural, earthy aesthetics:
- Deep Maroon (#893941)
- Dusty Rose (#CB7885)
- Light Green (#D4D994)
- Olive Green (#5E6623)
- Cream (#F5F5F0)
